## installation
```bash
npm install -S @jswork/next-cdb-case
```

## usage
```js
import '@jswork/next-cdb-case';

const res = nx.cdbCase('玛鲁ABC亲子乐园 V1.0.0（全能版）L1');
// 玛鲁ABC亲子乐园 V1.0.0(全能版)L1
```
